# 📊 API REST de Consulta de Vendas
API REST desenvolvida com Java e Spring Boot para consulta de vendas e geração de relatórios e sumários por vendedor.

Este projeto consiste na implementação de consultas com JPA / JPQL, incluindo filtros por data, busca por vendedor e paginação de resultados.

---

## 🧠 Conceitos aplicados

Este projeto utiliza conceitos importantes do desenvolvimento backend com Spring:

* Consultas JPQL com DTO Projection
* Paginação com Spring Data
* Relacionamento Many-to-One (Sale → Seller)
* Filtros dinâmicos em consultas
* Arquitetura em camadas

---

## 🛠 Tecnologias utilizadas
* Java 17
* Spring Boot 2.7.3
* Spring Data JPA / Hibernate
* Banco de dados H2
* Maven
* Postman
* JPQL
